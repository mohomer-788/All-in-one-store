ALL-in-ONE Store - No-Code E-Commerce Website

## Platform Used

Wix

## Project Description

ALL-in-ONE Store is a no-code e-commerce website created using Wix. The website allows customers to browse products, view prices and descriptions, add products to the shopping cart, and contact the store through an easy-to-use interface.

## Features Implemented

- Homepage
- Product Page
- About Page
- Contact Page
- Shopping Cart
- Responsive Design
- Product Images and Descriptions
